# **App Name**: MuskBlog

## Core Features:

- Blog Post Display: Display the title, content, and metadata (author, date) of the blog post.
- Content Formatting: Format the blog content with headings, paragraphs, images, and embedded links to ensure readability. Ensure links open in a new tab.
- Related Posts: Recommend related articles based on the current blog post's content and categories.
- AI-Powered Summarization: Provide a tool which offers a short, AI-generated summary of the blog post.

## Style Guidelines:

- Primary color: Electric Blue (#7DF9FF) to reflect innovation.
- Background color: Light Gray (#D3D3D3), a desaturated hue of the primary.
- Accent color: Teal (#008080) to complement the primary with a sophisticated tone.
- Body font: 'Inter', a sans-serif font to give a machined, neutral look.
- Headline font: 'Space Grotesk', a sans-serif font that pairs nicely with Inter.
- Use minimalist icons for categories, tags, and social sharing buttons.
- Employ a clean, responsive layout optimized for readability on all devices. Use a single-column layout for content and a sidebar for related posts and navigation.
